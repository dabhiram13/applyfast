# Applyfast Design System

## Brand
- Source of truth: `docs/APPLYFAST_BRAND_LOCK.md`.
- Logo: angular forward mark + Applyfast wordmark.
- Personality: premium, evidence-first, direct, trustworthy.

## Colors
- Lime: #B7F34A
- Charcoal: #111318
- Near black: #0D0F12
- Soft canvas: #F7F8F5
- Soft gray: #F3F4F6
- Muted text: #6B7280
- White: #FFFFFF

## Components
- Rounded cards
- High-contrast CTAs
- Sponsor confidence badges
- Recommendation chips
- Evidence lists
- Data tables
- Analyzer panels
- Tracker cards

## UI direction
The product should feel like a clean SaaS command center, not a flashy job board. Dark surfaces should be used for authority/data sections. Light surfaces should be used for work screens.
