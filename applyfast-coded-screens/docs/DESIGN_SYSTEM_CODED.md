# Applyfast coded design system

## Tokens

- Lime Green: `#B7F34A`
- Lime Strong: `#8BDC19`
- Charcoal: `#0D0F12`
- Dark Gray: `#1F2328`
- Muted Gray: `#9CA3AF`
- Soft Gray: `#F3F4F6`
- White: `#FFFFFF`

All tokens are defined in `app/globals.css` under `:root`.

## UI rules

1. Use lime green only for decision/action/high-confidence states.
2. Use dark charcoal for hero sections, sidebar, analytics, and high-emphasis surfaces.
3. Use white cards with subtle borders for productivity screens.
4. Preserve rounded cards, soft shadows, and high contrast.
5. Do not introduce random gradients or unrelated colors.
6. Do not invent a new logo.
